## How to run:


1. Clone the repo

```bash
  git clone https://github.com/uQwQu/tg-auth-bot.git
```

2. Start ngrok (or smth like that) to tunnel your local server

```bash
  ngrok http 8000
```

3. Create .env file based on .env.example

```bash
  cp .env.example .env
```

4. Set webhook

```bash
  ./webhook.sh
```

5. Run containers

```bash
  make build
```
